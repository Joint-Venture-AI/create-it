# Create It - Project Template Manager

Create projects from awesome templates with interactive UI

## Features

- **Interactive Template Selection**: Choose from a variety of project templates with an easy-to-use interface.
- **Customizable Options**: Tailor your project setup with customizable options during the creation process.
- **Quick Start**: Get your projects up and running in minutes with pre-configured templates.
- **Wide Range of Templates**: Access templates for different programming languages and frameworks.

Or install directly from the Visual Studio Code Marketplace:

1. Open Visual Studio Code.
2. Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window or by pressing `Ctrl+Shift+X`.
3. Search for "Create It - Project Template Manager".
4. Click the Install button to install the extension.

## How to Use

1. Open the Command Palette by pressing `Ctrl+Shift+P`.
2. Type `Create It: New Project` and select it.
3. Follow the prompts to select a template and customize your project.
4. Once completed, your new project will be created in the specified directory.
